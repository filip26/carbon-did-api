## Features

* DID, DID URL primitives
* DID Document API & Primitives
* DID Resolver API
  * [`did:key`](https://github.com/filip26/carbon-did-key) method

## Installation

### Maven

```xml
<dependency>
    <groupId>com.apicatalog</groupId>
    <artifactId>carbon-did</artifactId>
    <version>0.9.1</version>
</dependency>
```
